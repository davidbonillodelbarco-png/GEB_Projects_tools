The first part of the practice consisted on getting familiarized with github and Visual Studio Code, which would syncronize the changes made on the files with the repository. Then, a simple exercise was performed: to blink a LED using an ESP32 (IP:192.168.1.32) board programmed with PlatformIO and the Arduino framework inside Visual Studio Code. Through this activity, we had to verify that the pin of the LED was correctly specified in the source code. 

Once the hardware setup was connected (the endo-module board with an ESP32 and the PC control with roboDK program and the python scripts) and the LED correctly blinked on and off every second, we were asked to complete a series of tasks.
